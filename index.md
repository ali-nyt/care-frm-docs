## Getting started

This site uses [Just the Docs](https://github.com/pmarsceill/just-the-docs), a documentation theme for [Jekyll](https://jekyllrb.com), which is a static site generator with built-in support for GitHub Pages and a simplified build process.

### Setup:

1. ... setup intstructions TK
2. Run you local Jekyll server
```bash
$ jekyll serve
```
```bash
# .. or if you're using a Gemfile (bundler)
$ bundle exec jekyll serve
```
2. Point your web browser to [http://localhost:4000](http://localhost:4000)

This site is hosted on GitHub Pages, you can find more information about local setup [here](https://help.github.com/en/articles/setting-up-your-github-pages-site-locally-with-jekyll).
